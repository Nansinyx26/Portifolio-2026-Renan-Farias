Quiz do Sistema Solar em Realidade Virtual

Este projeto oferece uma experiência interativa e educativa de aprendizado sobre o Sistema Solar, utilizando a realidade virtual (VR). Através de um navegador web e um headset de VR (opcional), o usuário pode explorar os planetas em 3D e responder a perguntas sobre seus nomes.

A experiência também inclui um modo de realidade aumentada (AR), que permite visualizar os planetas em um ambiente real, utilizando a câmera do dispositivo.

Funcionalidades

  • Exploração 3D: Visualize os planetas em um ambiente virtual em 3D.
  • Quiz: Responda a perguntas sobre os nomes dos planetas.
  • Realidade Aumentada: Veja os planetas em um ambiente real, utilizando a câmera do seu dispositivo.
  • Marcador Visual: Utilize o marcador visual para iniciar a experiência de realidade aumentada.

Controles e Como Usar

Controles Básicos

  • W: Aproximar
  • S: Afastar

Como Executar

  1. Abra o arquivo index.html em um navegador moderno (Chrome, Firefox, etc.).
  2. Modo VR: Se estiver usando um headset de VR, siga as instruções do seu navegador para entrar no modo VR.
  3. Modo AR: Aponte a câmera do seu dispositivo para o marcador visual (imagem marker.jpg) para ativar o modo de realidade aumentada.

Modo de Realidade Aumentada (AR)

Para utilizar o modo AR, siga estes passos:

  1. Imprimir o Marcador: Imprima a imagem do marcador "Hiro" em alta qualidade.
  2. Aponte a Câmera: Aponte a câmera do dispositivo para o marcador impresso. O aplicativo reconhecerá o marcador e sobreporá os planetas.

Dicas: Utilize uma impressora de alta resolução e garanta que o ambiente esteja bem iluminado para melhor reconhecimento.

Tecnologias Utilizadas

  • A-Frame: Framework de código aberto para criar experiências de VR e AR para a web.
  • AR.js: Biblioteca JavaScript para realidade aumentada na web.
  • HTML, CSS e JavaScript: Stack padrão de desenvolvimento web.

Personalização

  • Planetas: Adicione ou remova planetas, customize suas cores, tamanhos e posições.
  • Perguntas: Modifique as perguntas do quiz e suas respostas corretas.
  • Marcador: Crie um novo marcador visual com um design personalizado.
