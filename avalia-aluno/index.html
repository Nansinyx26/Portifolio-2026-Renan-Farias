Sistema de Avaliação com Estrelas

Um componente de interface moderno e interativo para avaliação de desempenho estudantil, utilizando feedback visual imediato através de um sistema de estrelas.

Funcionalidades

  • Feedback Visual: Micro-interações e animações ao selecionar a pontuação.
  • UX Otimizada: Interface limpa e fácil de entender para professores.
  • Adaptabilidade: Componente modular que pode ser integrado em diferentes sistemas.
  • Responsividade: Funciona perfeitamente em dispositivos touch.

Tecnologias

  • JavaScript (DOM Manipulation)
  • CSS3 (Animations & Flexbox)
  • HTML5
  • Font Awesome Icons
