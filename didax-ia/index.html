Didax-IA

Didax-IA é uma plataforma educacional inovadora que integra Inteligência Artificial para auxiliar professores e alunos no processo de ensino-aprendizagem. A plataforma oferece ferramentas inteligentes para personalização de conteúdo e suporte pedagógico.

Funcionalidades Inteligentes

  • Assistente Pedagógico: IA treinada para auxiliar na criação de planos de aula e materiais didáticos.
  • Personalização: Conteúdo adaptativo com base no progresso do aluno.
  • Análise de Dados: Insights sobre o desempenho da turma para o professor.
  • Interação Dinâmica: Chat inteligente para tirar dúvidas recorrentes.

Tecnologias

  • JavaScript (Vanilla)
  • Integração com APIs de IA (OpenAI/Google Gemini)
  • CSS Moderno & Design Responsivo
  • Vercel for Deployment
